My World Heritage

Version -

Track World heritage visits and travel with a maintained UNESCO site catalogue, while keeping personal data under your control. A periodic opt-in statistical submission can be enabled to track usage interest; no personally identifiable information is submitted, and a pseudonomous identity links submissions from the same client.

Where data is stored

  • Site catalogue in use: data/current/unesco_official_sites.geojson.
  • Profile, visits and usage counters: browser localStorage (persists across browser restarts).
  • Logout state: in-memory on the current page only (no browser storage key).
